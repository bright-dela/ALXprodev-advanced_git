# ALXprodev-advanced_git

This repository demonstrates the **GitFlow workflow** and best practices for managing a collaborative Git project. The repository includes branches for features, releases, and hotfixes following the GitFlow model.

## Branch Structure

- `main` – Production-ready code
- `develop` – Active development branch
- `feature/*` – Branches for new features
- `release/*` – Branches for release preparation
- `hotfix/*` – Branches for urgent bug fixes
